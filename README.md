# Mobile-App-Wireframing
Created low-fidelity mobile app wireframes for an Exam Stress Management application using Figma, focusing on layout structure, navigation flow, and usability.

This project contains mobile app wireframes designed using Figma for an Exam Stress Management application.

## Features
- Login screen layout
- Dashboard wireframe
- Study planner screen
- Stress tracking interface
- User navigation flow

## Tool Used
- Figma

## Project Type
Mobile App Wireframing
